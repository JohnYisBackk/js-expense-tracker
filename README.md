Track your income and expenses, monitor your balance in real time, and store your data locally in the browser.

## ✨ Features

- ➕ Add income and expense transactions
- 🗑️ Delete transactions
- 📊 Real-time balance calculation
- 💰 Separate totals for income and expenses
- 💾 Data persistence using **localStorage**
- 📱 Fully responsive design
- 🌙 Modern dark UI

---

## 🧠 How It Works

The application follows a simple state-driven architecture:

1. **Transactions Array**
   - All data is stored in a central `transactions` array

2. **Add Transaction**
   - User inputs are validated
   - New transaction is added to the array

3. **Render UI**
   - Transactions are dynamically rendered into the DOM

4. **Update Summary**
   - Income, expenses, and balance are recalculated

5. **Persistence**
   - Data is saved to `localStorage`
   - Automatically loaded on page refresh
